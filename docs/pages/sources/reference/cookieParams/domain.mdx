# domain <Badge type={'property'}>Property</Badge> <Badge type={'modifier'}>Optional</Badge>

Specifies the domain for which the cookie is valid

## API

```ts copy
domain?: string;
```

## Example

```ts copy
import {cookiePart} from '@bitfiber/rx';

const token = cookiePart('token');

token.set({
  value: 'token-hash',
  domain: '.example.com',
});
```
