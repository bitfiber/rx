# observe <Badge type={'method'}>Method</Badge>

Returns an observable that will emit value changes for a specific key

`@param key: string {:ts}`  
The key for which to observe value changes

`@returns Observable<T> {:ts}`  
An observable that will emit value changes for a specific key

## API

```ts copy
observe(key: string): Observable<T>;
```

## Example

```ts copy
import {tap} from 'rxjs';
import {memoryStorage} from '@bitfiber/rx';

const ms = memoryStorage();
ms.observe('key').pipe(tap(v => console.log(v))).subscribe();
```
