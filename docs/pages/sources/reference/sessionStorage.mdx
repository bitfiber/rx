# SessionStorage <Badge type={'class'}>Class</Badge>

Implements the [KeyValueSource](/sources/reference/keyValueSource) interface, allowing interaction
with the browser's session storage using key-value semantics. It provides methods for retrieving,
setting, observing, and removing key-value pairs stored in session storage

`@template T = any {:ts}`  
The type of the value stored in session storage. Defaults to `any`

## API

```ts copy
class SessionStorage<T = any> implements KeyValueSource<T> {
  get(key: string): T;
  set(key: string, value: T): void;
  remove(key: string): void;
  observe(key: string): Observable<T>;
  destroy(): void;
}
```

## Example

```ts copy
import {SessionStorage} from '@bitfiber/rx';

const ss = new SessionStorage();
```
