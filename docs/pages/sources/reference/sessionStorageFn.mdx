# sessionStorage <Badge type={'function'}>Function</Badge>

Creates and returns a singleton instance of `SessionStorage` and ensures that only one instance
is created. If the instance already exists, it returns the existing instance

`@template T = any {:ts}`  
The type of the value stored in session storage. Defaults to `any`

`@returns SessionStorage<T> {:ts}`

See also: [SessionStorage](/sources/reference/sessionStorage)

## API

```ts copy
function sessionStorage<T = any>(): SessionStorage<T>;
```

## Example

```ts copy
import {sessionStorage} from '@bitfiber/rx';

const ss = sessionStorage();
```
